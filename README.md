# Chatbot


# How to run?
### STEPS:

Clone the repository

```bash
Project repo: https://github.com/ 
```

### STEP 01- Create a conda environtment after opening the repository

```bash
conda create -n cbotenv python=3.10 -y
```

```bash
conda activate cbotenv
```

### STEP 02- install the requirements
```bash
pip install -r requirements.txt
```
